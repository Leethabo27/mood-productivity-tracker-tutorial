About This Project

This project is a simple, responsive dashboard showcasing HTML, CSS, JavaScript, APIs, modules, localStorage, and Chart.js.

  • Live date & time (local timezone)
  • Responsive navigation with a mobile toggle
  • Theme switch (light/dark)
  • Animated gradient background + dynamic images
  • Quotes API integration
  • Notes saved to localStorage
  • Mood tracker + Chart.js visualization

Built to be user-friendly on all devices.
